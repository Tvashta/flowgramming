Problem ID - Problem Name

Problem Description

You are given a string and your task is to swap cases. In other words, convert all lowercase letters to uppercase letters and vice versa.
For Example:
www.AmritaUniversity.com → WWW.aMRITAuNIVERSITY.COM Pythonist 3 → pYTHONIST 3

Input Format

A single line containing a string S.

Output Format

Print the modified string S.

Constraints

0<=len(S)<=1000

Author(s)

TheTiredTrinity

Points

10

Memory Limit

64M

Time Limit

10.0s

Sample Input

AmritaUniversity.com presents "Pythonist 3".

Sample Output

aMRITAuNIVERSITY.COM PRESENTS "pYTHONIST 3".

Explanation

The cases of all the letters in the string have been swapped
